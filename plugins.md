# Plugins

## Projektmanagement

| Plugin/Skill | Mehrwert |
|---|---|
| [grill-me](./projektmanagement/skills/grill-me/) | Stellt hartnäckige Rückfragen, bis eine Idee vollständig durchdacht ist. |
| [write-a-prd](./projektmanagement/skills/write-a-prd/) | Verwandelt eine rohe Idee in ein strukturiertes PRD als Jira- oder GitHub-Issue. |
| [prd-to-issues](./projektmanagement/skills/prd-to-issues/) | Zerlegt ein PRD in unabhängig umsetzbare Tickets nach dem Vertical-Slice-Prinzip. |

## Entwicklung
